1. Voer de productnaam in zoals op de verpakking (bijv. "Moroccanoil – Treatment – 100 ml")

5. Controleer en pas aan waar nodig in de verschillende tabbladen

Afbeeldingen naar Magento pushen

        
Upload de afbeeldingen naar het gekoppelde Magento product

      


      


      

      

    


    
    

      

        
Workflow

        1. Zoek en koppel een bestaand Magento product

        2. Upload je productafbeeldingen

        3. Selecteer de hoofdafbeelding (★)

        4. Optioneel: verwijder achtergronden en/of resize naar 1200×1200

        5. Push de afbeeldingen naar Magento


        Afbeeldingsrollen in Magento

        • ★ Hoofdafbeelding = base image, small image én thumbnail tegelijk

        • Overige afbeeldingen worden als gallery images toegevoegd


        Tip: Verwijder eerst de achtergrond, resize daarna voor het beste resultaat.

Attribuut Mapping

        
Controleer of de attribute codes overeenkomen met jouw Magento

      

      

        
Tool veld → Magento attribute_code

        ──────────────────────────────

        Productnaam → name

        Short description → short_description

        Omschrijving HTML → description

        Meta titel → meta_title

        SEO beschrijving → meta_description

        URL key → url_key

        Ingrediënten → ingredients

        Merk → brand

        Lijn → product_line

        Producttype → product_type_custom

        Inhoud → content_size

        Ik heb → ik_heb

        Werking → werking

        Vrij van → vrij_van

        

        ⚠ Pas attribute codes aan in productopvoer-api.php → $haarshopAttrs
