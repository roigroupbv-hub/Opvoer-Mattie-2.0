# Haarshop.nl – Productopvoer Tool

Interne tool voor het gestructureerd opvoeren van producten in Magento voor [Haarshop.nl](https://www.haarshop.nl).

## Wat doet deze tool?

De Productopvoer Tool stroomlijnt het invoeren van nieuwe producten door:

- **Gestructureerde invoervelden** voor alle benodigde productdata (naam, merk, lijn, inhoud, ingrediënten, etc.)
- **SEO-optimalisatie** met automatische generatie van meta titles, meta descriptions en URL keys
- **HTML-output** voor short descriptions en productomschrijvingen, direct kopieerbaar naar Magento
- **Validatie & voortgangsindicator** om te zorgen dat alle verplichte velden correct zijn ingevuld
- **Afbeeldingenbeheer** voor het toevoegen en ordenen van productfoto's
- **Clipboard-functionaliteit** om velden snel naar Magento te kopiëren

## Gebruik

Open `index.html` in je browser — de tool werkt volledig offline en heeft geen server nodig.

## Technologie

- HTML / CSS / JavaScript (single-file, zelfstandig)
- Geen externe dependencies
- Offline bruikbaar
